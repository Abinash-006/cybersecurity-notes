SQL Injection and XSS

## SQL Injection (SQLi)
SQL Injection is a vulnerability where attacker input affects database queries.

### Impact
- Read sensitive data
- Bypass authentication
- Modify database

---

## Cross-Site Scripting (XSS)
XSS allows attackers to inject JavaScript into web pages.

### Types
- Stored XSS
- Reflected XSS
- DOM-based XSS

### Impact
- Steal cookies
- Session hijacking
- Fake user actions

---

## Why These Matter
SQLi and XSS are part of OWASP Top 10.
They are common and dangerous web vulnerabilities.
